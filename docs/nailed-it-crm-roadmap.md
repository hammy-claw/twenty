# Nailed It CRM Roadmap

## Research summary: how Houzz is being used today

Based on prior session evidence and public Houzz Pro docs, Nailed It appears to be using Houzz as a project operating system, not just a lead source.

Observed / inferred sources of truth:
- lead intake and qualification
- estimate/proposal creation and versioning
- project dashboard and status visibility
- selections and allowance tracking
- financial documents and billing history
- schedule and milestone visibility
- client communication history
- change orders and approvals

Important signals from prior sessions:
- Houzz Pro dashboard showed Nailed It Developments INC.
- 41 projects
- 224 financial documents
- Estimate creation flow supports:
  - from scratch
  - upload
  - template
  - AI-assisted creation
- Public feature pages show the connected modules:
  - estimates
  - proposals
  - selections
  - takeoffs
  - schedule
  - financial docs

## Replacement strategy

The CRM should replace Houzz in the order of business value:
1. leads and qualification
2. estimates and approvals
3. project handoff
4. selections and change orders
5. tasks and production planning
6. vendor and subcontractor tracking
7. AI-assisted operations
8. client portal / dashboard

## Houzz feature surface observed in the live account

Core surfaces currently visible in the Nailed It Houzz Pro account:
- Leads / CRM with stages, filters, tags, managers, and lead-source tracking
- Contacts / company directory
- Projects dashboard and per-project overview pages
- Project planning: estimates, takeoffs, 3D floor plans, mood boards, selection boards
- Project management: files & photos, schedule, tasks & punchlist, client dashboard, daily logs, time & expenses
- Finance: invoices, purchase orders, change orders, retainers & credits, financial overview
- Company library: financial reports, templates, categories, my items, catalogs
- Marketing surfaces: Houzz profile, contact form, website
- Platform surfaces: integrations, notifications, communication, cloud storage, account info, release notes, help/training

Observed estimate editor features worth matching or exceeding:
- draft/send/preview/download/share
- comments and automations
- create invoice from estimate
- items table with line-item editing
- assembly building
- templates / recent estimates / catalogs / my items
- schedule creation from estimate
- AI-assisted scheduling and estimate assistance

## V1 system rules

- Premium, not cheap
- Simple, not bloated
- Phone-first lead qualification before consultation
- Scope discipline: anything not explicitly in the approved quote becomes a change order
- Procurement remains separate from production planning
- Production tools should stay high-level and easy to maintain
